# markdown-gen (mdg)

+ A command line tool that can generate markdown using some options.

### Usage

```bash
$ mdg generate --help

# Usage: mdg generate [OPTIONS] INPUT_FILE

#   Generate markdown by options.

# Options:
#   -o, --output TEXT  [required]
#   --force
#   --center_image
#   --en_punctuation
#   --katex_image
#   --help             Show this message and exit.
```
